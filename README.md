### 关于rtsp协议

互联网TCP/IP协议中五层体系结构中的应用层协议,该协议一般用于直播
特点:
1）是流媒体协议。
2）RTSP协议是共有协议，并有专门机构做维护。.
3）RTSP协议一般传输的是 ts、mp4 格式的流。
4）RTSP传输一般需要 2-3 个通道，命令和数据通道分离。

第二点使该协议有很好的通用性.海康等相机支持该协议.
